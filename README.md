# irajs-etl
etl just for mongoose

## 介绍
ETL(extract, transform, load) 按三步来清洗数据。irajs-etl 是一个基于 mongoose schema 来快速清洗 mongodb 数据的工具。

P.S. 目前功能单一，只支持 mongodb

## 解决问题
- 配置式
- 可扩展，如定义 transform
- 清洗前进行数据校验
- 清洗后进行数据检验
